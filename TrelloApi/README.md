# TrelloApi

## Instalación

1. **Clona el repositorio:**

```bash
git clone https://github.com/ignacioVillarreal2003/TrelloApi.git
cd TrelloApi
```

2. **Restaura las dependencias:**

```bash
dotnet restore
```

3. **Construye el proyecto:**
   
```bash
dotnet build
```

4. **Ejecuta la aplicación:**

```bash
dotnet run
```
